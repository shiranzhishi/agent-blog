## 快速开始

### 1. 启动博客服务器
```bash
# 启动后端API
cd server
npm run dev

# 启动前端（另一个终端）
npm run dev
```

### 2. 获取认证令牌

**方法A：通过前端界面**
1. 访问 http://localhost:5173
2. 登录你的账户
3. 在浏览器开发者工具 → Application → Local Storage 中找到 `token`

**方法B：直接API调用**
```bash
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"your-email@example.com","password":"your-password"}'
```

### 3. 在Kiro中使用

MCP服务器已经配置完成，现在你可以直接在对话中使用：

#### 设置认证
```
请帮我设置博客认证令牌：eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

#### 创建并发布文章
```
请帮我创建一篇文章：
标题：Vue 3 Composition API 深度解析
内容：在现代前端开发中，Vue 3的Composition API为我们提供了更灵活的组件逻辑组织方式...
立即发布：true
```

#### 管理现有文章
```
请显示我的所有草稿文章，然后发布ID为2的文章
```

## 故障排除

### 常见问题

**Q: MCP服务器无法连接**
A: 检查 `.kiro/settings/mcp.json` 配置和文件路径

**Q: 认证失败**
A: 确保JWT令牌有效且未过期

**Q: API连接失败**
A: 确保博客服务器在 http://localhost:3000 运行
